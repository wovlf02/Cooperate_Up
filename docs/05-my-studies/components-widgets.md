# 🧩 사이드바 위젯 컴포넌트

## 개요

스터디 대시보드 우측 사이드바에 표시되는 위젯들입니다.

---

## 컴포넌트 목록

| 컴포넌트 | 설명 |
|---------|------|
| MemberWidget | 멤버 위젯 |
| NextEventWidget | 다음 일정 위젯 |
| FileWidget | 파일 위젯 |
| TaskWidget | 할일 위젯 |
| QuickActionsWidget | 빠른 액션 |

---

## MemberWidget

### 설명

스터디 멤버 현황을 표시합니다.

### 표시 정보

- 총 멤버 수
- 온라인 멤버 수
- 멤버 아바타 (최대 5명)
- "+N명" 표시

---

## NextEventWidget

### 설명

가장 가까운 일정을 표시합니다.

### 표시 정보

- 일정 제목
- 날짜/시간
- 장소
- D-day 표시

---

## FileWidget

### 설명

최근 파일 업로드 현황을 표시합니다.

### 표시 정보

- 총 파일 수
- 최근 업로드 3개
- 전체 보기 링크

---

## TaskWidget

### 설명

스터디 할일 현황을 표시합니다.

### 표시 정보

- 진행 중인 할일 수
- 마감 임박 할일
- 완료율 진행바

---

## QuickActionsWidget

### 설명

자주 사용하는 액션 버튼들입니다.

### 액션

| 액션 | 설명 |
|------|------|
| 화상통화 시작 | 화상통화 바로 시작 |
| 공지 작성 | 새 공지 작성 |
| 파일 업로드 | 빠른 업로드 |
| 일정 추가 | 새 일정 추가 |
