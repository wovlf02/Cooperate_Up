# 📱 파일 화면

## 접근 경로

| 항목 | 값 |
|------|-----|
| **URL** | `/my-studies/[studyId]/files` |

---

## 기능

### 파일 목록

| 컬럼 | 설명 |
|------|------|
| 아이콘 | 파일 타입별 아이콘 |
| 파일명 | 파일 이름 |
| 크기 | 파일 크기 |
| 업로더 | 업로드한 사람 |
| 날짜 | 업로드 날짜 |
| 다운로드 | 다운로드 횟수 |

### 파일 액션

| 액션 | 설명 |
|------|------|
| 다운로드 | 파일 다운로드 |
| 미리보기 | 이미지/PDF 미리보기 |
| 삭제 | 본인 업로드 파일만 |

### 업로드 제한

| 항목 | 제한 |
|------|------|
| 최대 크기 | 10MB |
| 허용 형식 | 이미지, PDF, 문서, 압축 |
