## 기능

### 캘린더 뷰

| 요소 | 설명 |
|------|------|
| 월 네비게이션 | 이전/다음 월 이동 |
| 일정 표시 | 날짜에 점으로 표시 |
| 날짜 선택 | 클릭 시 해당 일 일정 |

### 일정 표시

| 요소 | 설명 |
|------|------|
| 색상 | 일정 구분 색상 |
| 시간 | 시작-종료 시간 |
| 제목 | 일정 제목 |
| 장소 | 모임 장소 |

### 일정 추가

| 필드 | 설명 |
|------|------|
| 제목 | 일정 이름 |
| 날짜 | 일정 날짜 |
| 시작/종료 시간 | 시간 선택 |
| 장소 | 모임 장소 (선택) |
| 색상 | 일정 색상 |
