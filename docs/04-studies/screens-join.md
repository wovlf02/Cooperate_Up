# 📱 스터디 가입 화면

## 접근 경로

| 항목 | 값 |
|------|-----|
| **URL** | `/studies/[studyId]/join` 또는 모달 |

---

## 폼 필드

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| introduction | 텍스트 | - | 자기소개 |
| motivation | 텍스트 | - | 가입 동기 |
| level | 선택 | - | 현재 수준 |

---

## 수동 승인 스터디

수동 승인(`autoApprove: false`)인 경우:
- 가입 신청 후 PENDING 상태
- 관리자 승인 필요
- 승인 시 알림 전송
