# 🚀 CoUp 프로젝트 소개

## 📋 개요

CoUp(Cooperate Up)은 **스터디 그룹 협업 플랫폼**으로, 함께 공부하는 사람들이 효율적으로 소통하고 학습 활동을 관리할 수 있도록 돕는 웹 애플리케이션입니다.

---

## 🎯 서비스 목적

### 해결하고자 하는 문제
1. **스터디 관리의 어려움**: 스터디 일정, 과제, 멤버 관리가 여러 도구에 분산
2. **소통의 비효율**: 카카오톡, 노션, 줌 등 여러 플랫폼을 오가며 소통
3. **진행 상황 파악 불가**: 누가 어디까지 했는지 한눈에 보기 어려움
4. **자료 관리 혼란**: 스터디 자료가 여기저기 흩어져 있음

### CoUp의 해결책
- **올인원 플랫폼**: 채팅, 일정, 과제, 파일을 한 곳에서
- **실시간 협업**: Socket.io 기반 실시간 채팅 및 알림
- **체계적인 관리**: 스터디별 대시보드, 태스크 관리, 출석 체크
- **그룹 기능**: 스터디 내 소그룹 구성 및 관리

---

## 👥 대상 사용자

| 사용자 유형 | 설명 | 주요 활동 |
|-------------|------|----------|
| **스터디 참여자** | 스터디에 가입하여 참여하는 일반 사용자 | 채팅, 과제 확인, 일정 확인 |
| **스터디 운영자** | 스터디를 생성하고 관리하는 리더 | 멤버 관리, 과제 배정, 공지 작성 |
| **시스템 관리자** | 전체 플랫폼을 관리하는 관리자 | 사용자 관리, 신고 처리, 통계 확인 |

## ✨ 핵심 기능

### 1. 스터디 관리
- **스터디 생성/참여**: 스터디 만들기, 검색, 가입 신청
- **멤버 관리**: 가입 승인, 역할 부여, 강퇴
- **설정**: 스터디 정보, 공개/비공개 설정

### 2. 실시간 채팅
- **그룹 채팅**: 스터디 전체 채팅방
- **소그룹 채팅**: 그룹별 별도 채팅방
- **실시간 알림**: 새 메시지 알림

### 3. 태스크(과제) 관리
- **과제 생성**: 제목, 설명, 마감일 설정
- **담당자 지정**: 개인별 또는 그룹별 배정
- **진행 상황**: TODO → IN_PROGRESS → DONE

### 4. 일정 관리
- **캘린더**: 스터디 일정 캘린더 뷰
- **이벤트**: 정기 모임, 특별 일정 등록
- **알림**: 일정 전 알림

### 5. 대시보드
- **개인 대시보드**: 내 스터디 현황, 다가오는 일정
- **스터디 대시보드**: 스터디별 활동 현황
- **관리자 대시보드**: 전체 통계, 신고 관리

### 6. 알림 시스템
- **실시간 알림**: 새 메시지, 과제 배정, 가입 승인 등
- **알림 설정**: 알림 종류별 ON/OFF
- **읽음 처리**: 개별/전체 읽음 처리

### 7. 화상 통화
- **비디오 미팅**: 스터디 멤버 간 화상 회의
- **시그널링 서버**: WebRTC 기반 P2P 연결

## 💡 서비스 특징 및 차별점

### 🎨 직관적인 UI/UX
- 깔끔하고 현대적인 디자인
- 반응형 레이아웃 (데스크톱/모바일)
- 다크모드 지원

### ⚡ 빠른 성능
- Next.js App Router 기반 서버 사이드 렌더링
- TanStack Query 캐싱으로 빠른 데이터 로딩
- 최적화된 번들 사이즈

### 🔒 안전한 시스템
- NextAuth 기반 안전한 인증
- 역할 기반 접근 제어 (RBAC)
- XSS, CSRF 방지

### 🔄 실시간 협업
- Socket.io 기반 실시간 통신
- Redis 어댑터로 스케일 아웃 지원
- 낙관적 업데이트로 부드러운 UX
