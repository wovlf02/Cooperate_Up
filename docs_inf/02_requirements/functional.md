## 1. 사업장 관리 (FR-01 ~ FR-04)

### FR-01: 사업장 등록
| 항목 | 내용 |
|------|------|
| **설명** | 사업주가 새 사업장을 등록 (사업자등록번호 검증 필수) |
| **필수입력** | 사업장명, 도로명 주소(검색), 사업자등록번호, 대표자명, 업종, 허용 반경 |
| **검증** | 국세청 API를 통한 사업자등록번호 진위확인 (사업자상태, 대표자명, 업종 확인) |
| **기능** | 도로명 주소 검색 API, 주소→GPS 좌표 자동 변환, 사업자등록번호 실시간 검증 |
| **출력** | 검증 완료 후 사업장 생성, 고유 초대 코드 발급, GPS 좌표 자동 저장 |
| **제한** | 사업자등록번호 검증 실패 시 사업장 등록 불가 |
| **권한** | 사업주 전용 |
| **우선순위** | 필수 |

### FR-01-1: 사업자등록번호 검증
| 항목 | 내용 |
|------|------|
| **설명** | 국세청 API를 통한 사업자등록번호 진위확인 |
| **검증항목** | 사업자등록번호 유효성, 사업자 상태(계속/휴업/폐업), 대표자명 일치 여부 |
| **API** | 국세청 사업자등록 상태조회 API (공공데이터포털) |
| **조건** | 사업자 상태가 '계속사업자'인 경우에만 등록 허용 |
| **결과** | 검증 성공/실패 여부, 실패 시 사유 표시 |
| **우선순위** | 필수 |

### FR-02: 사업장 정보 수정
| 항목 | 내용 |
|------|------|
| **설명** | 사업주가 사업장 정보를 수정 |
| **입력** | 사업장명, 주소, GPS 좌표, 허용 반경 |
| **권한** | 사업주 전용 |
| **우선순위** | 필수 |

### FR-03: 사업장 GPS 좌표 설정
| 항목 | 내용 |
|------|------|
| **설명** | 도로명 주소 기반 GPS 좌표 자동 설정 (지오코딩) |
| **기능** | 네이버 지오코딩 API 활용 (무료), 좌표 정확도 검증, 현재 위치 미세 조정 (선택), 지도에서 수동 조정 (선택) |
| **정확도** | 지오코딩 결과 + 보조 검증으로 오차 범위 최소화 |
| **권한** | 사업주 전용 |
| **우선순위** | 필수 |

### FR-04: 초대 코드 관리
| 항목 | 내용 |
|------|------|
| **설명** | 직원 초대용 아이디 기반 초대 시스템 |

| **권한** | 사업주 전용 |
| **우선순위** | 필수 |

### FR-04-1: 사업장 근무 시간 설정
| 항목 | 내용 |
|------|------|
| **설명** | 사업장별 기본 근무 시간 설정 (조기 출근 급여 과지급 방지) |
| **기능** | 사업장 기본 운영 시간 설정, 조기 출근 시 설정된 시업 시간부터 급여 계산, 근무 시간 외 출근 체크 시 안내 메시지 |
| **입력** | 시업 시간, 종업 시간, 요일별 근무 시간 (선택) |
| **규칙** | 설정된 시업 시간 이전 출근 시 → 시업 시간부터 급여 계산, 설정된 종업 시간 이후 퇴근 시 → 연장근로로 계산 |
| **권한** | 사업주 전용 |
| **우선순위** | 필수 |

## 2. 사용자 관리 (FR-05 ~ FR-09)

### FR-05: 회원가입 (직원)
| 항목 | 내용 |
|------|------|
| **설명** | 직원용 일반 회원가입 |
| **필수입력** | 아이디, 비밀번호, 비밀번호 확인, 이름, 연락처, 이메일, 생년월일, 프로필 사진 |
| **검증** | 아이디 중복확인 (실시간), 비밀번호 일치 확인, 이메일 인증 (네이버 SMTP) |
| **비밀번호** | 영문, 숫자, 특수문자 포함 8자 이상, 비밀번호 강도 표시 |
| **출력** | 모든 검증 완료 후 계정 생성 |
| **제한** | 아이디 중복확인, 비밀번호 일치, 이메일 인증 미완료 시 회원가입 불가 |
| **우선순위** | 필수 |

### FR-05-1: 회원가입 (관리자 - 사업자 인증)
| 항목 | 내용 |
|------|------|
| **설명** | 관리자(사업주)용 회원가입 - 사업자 인증 필수 |
| **필수입력** | 아이디, 비밀번호, 비밀번호 확인, 이름, 연락처, 이메일, 생년월일, 프로필 사진, 사업자등록번호, 대표자명, 개업연월일 |
| **검증** | 아이디 중복확인 (실시간), 비밀번호 일치 확인, 이메일 인증 (네이버 SMTP), 국세청 사업자등록 상태조회 API를 통한 진위확인 |
| **비밀번호** | 영문, 숫자, 특수문자 포함 8자 이상, 비밀번호 강도 표시 |
| **검증항목** | 사업자등록번호 유효성, 대표자명 일치, 사업자 상태(계속/휴업/폐업), 개업연월일 일치 |
| **조건** | 사업자 상태가 '계속사업자'이고 대표자명이 일치하는 경우에만 회원가입 허용 |
| **출력** | 모든 검증 완료 후 관리자 계정 생성, 사업자 정보 저장 |
| **제한** | 아이디 중복확인, 비밀번호 일치, 이메일 인증, 사업자 인증 중 하나라도 미완료 시 회원가입 불가 |
| **우선순위** | 필수 |

### FR-06: 사업장 참여 (직원)
| 항목 | 내용 |
|------|------|
| **설명** | 사업주가 직원 아이디로 초대를 보내고, 직원이 승인하여 사업장에 참여 |
| **흐름** | 사업주가 직원 아이디 입력 → 초대 전송 → 직원에게 푸시 알림 → 직원이 초대 승인/거부 → 승인 시 사업장 참여 완료 |
| **입력** | (사업주) 직원 아이디 / (직원) 승인/거부 선택 |
| **출력** | 사업장 참여 완료, 사업주에게 결과 알림 |
| **권한** | 사업주: 초대 전송 / 직원: 초대 수락/거부 |
| **우선순위** | 필수 |

| **저장** | 로그인 상태 영구 저장 (AsyncStorage + SecureStorage) |
| **우선순위** | 필수 |

## 3. 출퇴근 관리 (FR-09 ~ FR-13)

### FR-09: GPS 기반 출근
| 항목 | 내용 |
|------|------|

| **출력** | 출근 시간 기록, 위치 저장, 급여 계산 기준 시간 표시 |

## 4. 급여 관리 (FR-14 ~ FR-20)

### FR-14: 직원별 시급 설정
| 항목 | 내용 |
|------|------|
| **설명** | 사업주가 직원별 개별 시급 설정 (근무자 수 제한 없음) |
| **기능** | 최저시급 미만 입력 시 알림창 표시 (최저시급 금액 안내, 그 이상 입력 필수 안내), 설정 불가 |
| **검증** | 2025년 최저시급 10,030원, 2026년 10,320원 등 연도별 자동 적용 |
| **알림** | "현재 최저시급은 {금액}원입니다. 최저시급 이상의 금액을 입력해주세요." |
| **권한** | 사업주 전용 |
| **우선순위** | 필수 |

### FR-15: 급여 자동 계산 (기본급)
| 항목 | 내용 |
|------|------|
| **설명** | 근무 시간 기반 기본 급여 자동 계산 |
| **규칙** | 기본급 = 근무 시간 × 시급 |
| **단위** | 분 단위 계산 |
| **산출식** | 급여 명세서에 상세 계산 산출식 자동 표시 |
| **우선순위** | 필수 |

### FR-15-1: 주휴수당 자동 계산
| 항목 | 내용 |
|------|------|
| **설명** | 근로기준법 기반 주휴수당 자동 계산 |
| **조건** | 주 15시간 이상 근무, 주 소정근로일 개근 |
| **규칙** | (1주 총 근무시간 ÷ 주 소정근로일수) × 시급 |
| **기준일** | 매주 일요일 기준 산정 |
| **우선순위** | 필수 |

### FR-15-2: 연장/야간/휴일 수당 자동 계산
| 항목 | 내용 |
|------|------|
| **설명** | 근로기준법 제56조 기반 가산수당 자동 계산 |
| **연장근로** | 1일 8시간, 1주 40시간 초과 시 통상임금의 50% 가산 |
| **야간근로** | 22:00~06:00 근무 시 통상임금의 50% 가산 |
| **휴일근로** | 휴일 근무 시 통상임금의 50% 가산 |
| **중복적용** | 휴일 야간 근로 = 100% 가산 |
| **우선순위** | 필수 |

### FR-15-3: 4대 보험 및 세금 자동 계산
| 항목 | 내용 |
|------|------|
| **설명** | 세무 연동 기반 공제액 자동 계산 |
| **국민연금** | 월 소득의 4.5% (근로자 부담분) |
| **건강보험** | 월 소득의 3.545% (근로자 부담분) |
| **장기요양** | 건강보험료의 12.95% |
| **고용보험** | 월 소득의 0.9% |
