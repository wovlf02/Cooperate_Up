# 회원가입 화면 (RegisterScreen) - Production Ready v2.0

## 개요

신규 사용자 계정 생성을 위한 회원가입 화면입니다.
토스, 카카오 등의 온보딩 UX를 참고하여 단계별로 직관적이고 세련된 디자인을 제공합니다.

---

## 🎨 디자인 원칙

### UX 목표

- **단계별 진행**: 복잡한 폼을 작은 단계로 분리하여 부담 감소
- **중앙 집중 레이아웃**: 각 단계별로 중앙 정렬로 집중도 향상
- **실시간 피드백**: 입력 즉시 유효성 검사 결과 표시
- **진행 상태 시각화**: 스테퍼로 현재 위치와 남은 단계 표시
- **부드러운 전환**: 단계 간 자연스러운 애니메이션

## 화면 흐름 (Step Flow)

```
┌─────────────────┐
│  역할 선택      │  Step 1

         │
```

```typescript
import React, { useState, useMemo } from 'react';

import { BasicInfoStep } from './steps/BasicInfoStep';
import { AccountStep } from './steps/AccountStep';
import { BusinessStep } from './steps/BusinessStep';
import { ProfileStep } from './steps/ProfileStep';
import { AgreementStep } from './steps/AgreementStep';

  const progress = state.currentStep / totalSteps;
  
  const renderStep = () => {
    switch (state.currentStep) {
      case 1: return <RoleStep {...{state, dispatch}} />;
      case 2: return <BasicInfoStep {...{state, dispatch}} />;
      case 3: return <AccountStep {...{state, dispatch}} />;

        : <ProfileStep {...{state, dispatch}} />;
      case 5: return state.role === 'admin'
        ? <ProfileStep {...{state, dispatch}} />
        : <AgreementStep {...{state, dispatch}} />;
      case 6: return <AgreementStep {...{state, dispatch}} />;
      case 'complete': return <CompleteStep name={state.name} />;
    }
  };
  
  const isStepValid = useMemo(() => {
    // 각 Step별 유효성 검사
  }, [state]);
  
  return (
    <View style={styles.container}>
      {state.currentStep !== 'complete' && (
        <>
          <Header
            showBack
            onBack={handleBack}
            rightElement={<CloseButton onPress={handleClose} />}
          />

            progress={progress}

            totalSteps={totalSteps}
          />

        </ScrollView>
      </KeyboardAvoidingView>
      
      {state.currentStep !== 'complete' && (
        <View style={[styles.footer, { paddingBottom: insets.bottom + spacing.space4 }]}>
          <PrimaryButton

            onPress={handleNext}
            disabled={!isStepValid}
```
