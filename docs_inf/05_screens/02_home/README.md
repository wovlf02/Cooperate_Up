## 사업장 컨텍스트

홈 화면은 **현재 선택된 사업장** 기준으로 데이터를 표시합니다.
- 사업장 선택 후 진입
- 상단에 현재 사업장 이름 표시
- 다른 사업장으로 전환 가능

---

## 역할별 차이점

### 직원 홈
- 오늘의 출퇴근 상태 및 빠른 버튼
- 이번 달 예상 급여
- 체크리스트 진행률
- 최근 공지사항

### 사업주 홈
- 현재 출근 직원 현황
- 승인 대기 요청 목록
- 이번 달 전체 급여 현황
- 체크리스트 모니터링 요약
- 최근 공지사항 + 새 공지 작성

---

## 공통 컴포넌트

- `WorkplaceHeader`: 사업장 이름 및 전환 버튼
- `GreetingSection`: 인사말 및 날짜
- `QuickActionButton`: 빠른 액션 버튼
- `SummaryCard`: 요약 정보 카드
- `AnnouncementPreview`: 공지사항 미리보기

---

## 실시간 업데이트

홈 화면의 주요 데이터는 Firestore 실시간 구독을 통해 자동 업데이트됩니다.

| 항목 | 업데이트 |
|------|---------|
| 출퇴근 상태 | 실시간 |
| 체크리스트 진행률 | 실시간 |
| 공지사항 | 실시간 |
| 급여 정보 | 화면 진입 시 |
