```
10_contract/

```

---

## 📱 화면 목록

| 화면 | 파일명 | 권한 | 설명 |
|------|--------|------|------|
| 계약서 목록 | ContractListScreen | 공통 | 계약서 목록 조회 |
| 계약서 상세 | ContractDetailScreen | 공통 | 계약서 내용 및 서명 확인 |
| 계약서 작성 | ContractCreateScreen | 사업주 | 신규 계약서 작성 |
| 계약서 서명 | ContractSignScreen | 직원 | 계약서 서명 |

---

## 🎨 화면 플로우

### 사업주 플로우
```
계약서 목록 → 계약서 작성 → 직원 선택 → 내용 입력 → 사업주 서명 → 발송
     │
     └──→ 계약서 상세 → PDF 다운로드
```

### 직원 플로우
```
알림 수신 → 계약서 목록 → 계약서 상세 → 내용 확인 → 서명 → 완료
                                              │
                                              └──→ PDF 다운로드
```

---

## 📋 계약서 구성 항목

| 항목 | 설명 | 필수 |
|------|------|------|
| 근로자 정보 | 이름, 연락처 | ✓ |
| 근로 기간 | 시작일 ~ 종료일 | ✓ |
| 근무 장소 | 사업장 주소 | ✓ |
| 업무 내용 | 담당 업무 설명 | ✓ |
| 근무 시간 | 시작 ~ 종료, 휴게시간 | ✓ |
| 시급 | 시간당 급여 | ✓ |
| 급여 지급일 | 매월 지급일 | ✓ |
| 기타 사항 | 추가 약정 | |

---

## 🔖 계약서 상태

| 상태 | 설명 | 색상 |
|------|------|------|
| draft | 작성 중 | Gray |
| pending | 서명 대기 | Orange |
| signed | 서명 완료 | Green |
| expired | 만료됨 | Red |
