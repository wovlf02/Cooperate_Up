## 핵심 기능 (멀티테넌트)

### 사업주 - 체크리스트 관리
- **커스텀 체크리스트 생성**: 사업장별 맞춤 업무 목록 생성
- **카테고리 분류**: 오픈 업무, 마감 업무, 청소, 재고 등
- **시간대 설정**: 특정 시간에 수행할 업무 지정 (선택)
- **반복 요일 설정**: 매일/특정 요일만 표시
- **순서 지정**: 드래그 앤 드롭으로 순서 변경
- **활성화/비활성화**: 일시적으로 항목 숨기기

### 직원 - 체크리스트 수행
- 오늘 할당된 체크리스트 확인
- 원터치 체크 완료 + 햅틱 피드백
- 전체 진행률 실시간 표시
- 카테고리별/시간대별 필터링

### 사업주 - 모니터링
- 전체 직원 진행률 한눈에 확인
- 진행률 50% 미만 직원 경고 표시
- 직원별 상세 현황 조회
- 체크리스트 완료 내역 기록

| 컴포넌트 | 설명 |

| DraggableList | 드래그 앤 드롭 정렬 |

## 체크리스트 구조 예시

```
📂 오픈 업무 (09:00)
  ├── □ 매장 전등 켜기
  ├── □ 에어컨/난방 가동
  ├── □ 출입문 개방
  └── □ POS 부팅 확인

📂 청소
  ├── □ 바닥 청소
  ├── □ 화장실 청소
  └── □ 테이블 정리

📂 마감 업무 (22:00)
  ├── □ 매출 정산
  ├── □ 재고 확인
  ├── □ 전등/에어컨 끄기
  └── □ 출입문 잠금
```

## 진행률 상태

| 진행률 | 상태 | 색상 |
|--------|------|------|
| 70% 이상 | 양호 | success (green) |
| 50~69% | 보통 | primary (blue) |
| 50% 미만 | 주의 | warning (orange) |

---

## UX 특징

- 큰 체크박스: `ms(32) x ms(32)`
- 체크 시 햅틱 피드백 + scale 애니메이션
- 완료 시 배경색 변경 + 완료 시간 표시
- 카테고리별 접기/펼치기
- 사업주: 드래그 앤 드롭 항목 정렬
