### 기본 계산
```
일급 = 근무 시간 × 시급
월급 = 해당 월 일급 합계
```

### 20시 기준 계산
- 20시 이전 출근: 20시부터 계산
- 20시 이후 출근: 실제 출근 시간부터 계산

- 주휴 수당
