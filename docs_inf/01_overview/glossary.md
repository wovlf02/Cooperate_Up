# 용어 정의

## 1. 사용자

| 용어 | 정의 |
|------|------|
| **사업주 (Admin)** | 사업장을 운영하는 소상공인. 관리자 권한 보유 |
| **직원 (Employee)** | 사업장에서 근무하는 근로자. 일반 사용자 권한 |
| **테넌트 (Tenant)** | 멀티테넌트 구조에서 하나의 사업장 단위 |

## 2. 사업장

| 용어 | 정의 |
|------|------|
| **사업장 (Workplace)** | 직원이 근무하는 실제 장소. GPS 좌표 등록 |
| **허용 반경** | GPS 기반 출퇴근 인증 허용 거리 (기본 50m) |
| **사업장 코드** | 직원 초대 시 사용하는 고유 코드 |

## 3. 근태

| 용어 | 정의 |
|------|------|
| **출근 (Clock In)** | GPS 인증 후 출근 버튼으로 근무 시작 기록 |
| **퇴근 (Clock Out)** | GPS 인증 후 퇴근 버튼으로 근무 종료 기록 |
| **GPS 인증** | 사업장 허용 반경 이내 위치 검증 |
| **수동 근태 입력** | 버튼 미클릭 시 직접 시간 입력 (승인 필요) |
| **근태 수정 요청** | 기록된 시간 수정 요청 (승인 필요) |
| **근무 시간** | 출근~퇴근 시간 차이 |

## 4. 급여

| 용어 | 정의 |
|------|------|
| **시급** | 시간당 급여. 사업주가 직원별 개별 설정 |
| **최저시급** | 법정 최저시급 (2025: 10,030원, 2026: 10,320원) |
| **일급** | 하루 근무 시간 × 시급 |
| **월급** | 월간 근무 시간 × 시급 |
| **급여 명세서** | 근무 기록 및 급여 정리 PDF 문서 |

## 5. 체크리스트

| 용어 | 정의 |
|------|------|
| **업무 체크리스트** | 사업장별 커스텀 업무 목록 |
| **체크리스트 항목** | 개별 업무 태스크 |
| **카테고리** | 업무 분류 (예: 오픈 업무, 마감 업무) |
| **완료 체크** | 직원이 업무 완료 시 체크 |

## 6. 근로계약

| 용어 | 정의 |
|------|------|
| **전자근로계약서** | 앱 내에서 작성/서명하는 디지털 근로계약서 |
| **전자서명** | 계약서 동의를 위한 디지털 서명 |
| **계약 상태** | 대기/서명완료/만료 등의 계약 진행 상태 |
| **근로 조건** | 시급, 근무 시간, 근무 장소 등 계약 내용 |

## 7. 기술

| 용어 | 정의 |
|------|------|
| **React Native** | 크로스 플랫폼 모바일 앱 프레임워크 (0.83) |
| **JavaScript** | 프론트엔드/백엔드 공용 프로그래밍 언어 (ES6+) |
| **Spring Boot** | Java 기반 백엔드 프레임워크 (4.0.1) |
| **Spring Security** | Spring 기반 인증/인가 프레임워크 |
| **JDK** | Java Development Kit (21.0.8 LTS) |
| **PostgreSQL** | 오픈소스 관계형 데이터베이스 (18.1) |
| **Node.js** | JavaScript 서버 런타임 (24.11.0 LTS, 시그널링 서버) |
| **Socket.IO** | 실시간 양방향 통신 라이브러리 (WebSocket) |
| **AWS** | Amazon Web Services 클라우드 플랫폼 |
| **Amazon RDS** | AWS 관계형 데이터베이스 서비스 |
| **Amazon S3** | AWS 객체 스토리지 서비스 |
| **JWT** | JSON Web Token (인증 토큰, 직접 구현) |
| **Access Token** | 짧은 만료 시간의 인증 토큰 (API 호출용) |
| **Refresh Token** | 긴 만료 시간의 토큰 (Access Token 재발급용) |
| **BCrypt** | 비밀번호 해시 알고리즘 |
| **APNs** | Apple Push Notification service (iOS 푸시) |
| **Redux** | 상태 관리 라이브러리 |
| **React Query** | 서버 상태 관리 라이브러리 |
| **JPA** | Java Persistence API (ORM) |
| **멀티테넌트** | 하나의 앱에서 여러 사업장을 독립적으로 운영 |
| **시그널링 서버** | 실시간 통신을 위한 WebSocket 기반 서버 |
| **동적 할당** | 트래픽에 따라 서버 인스턴스 자동 스케일링 |
| **디바이스 토큰** | 푸시 알림 전송을 위한 디바이스 고유 식별자 |

## 8. 문서/파일

| 용어 | 정의 |
|------|------|
| **PDF 추출** | 급여명세서, 근로계약서를 PDF 파일로 저장 |
| **엑셀 추출** | 근무 기록을 Excel(.xlsx) 파일로 저장 |
| **첨부파일** | 공지사항, 채팅에 첨부하는 이미지/문서 |
