## 컴포넌트 구조

### 신고 목록 페이지

```jsx
<ReportsPage>
  ├── <PageHeader />
  ├── <ReportFilters />          # 필터 패널
  │   ├── 검색
  │   ├── 상태 필터
  │   ├── 유형 필터
  │   ├── 우선순위 필터
  │   └── 담당자 필터
  ├── <ReportList />             # 신고 목록
  │   ├── <ReportColumns />      # 테이블 헤더
  │   └── <ReportRow />[]        # 신고 행
  ├── <ReportBulkActions />      # 일괄 처리
  └── <Pagination />
</ReportsPage>
```

### 신고 상세 페이지

```jsx
<ReportDetailPage>
  ├── <PageHeader />
  ├── <ReportInfo />             # 신고 정보
  │   ├── 대상 정보
  │   ├── 신고 유형
  │   ├── 신고 사유
  │   └── 증거 자료
  ├── <ReporterInfo />           # 신고자 정보
  ├── <TargetInfo />             # 대상 정보 (이력)
  ├── <ReportActions />          # 처리 버튼
  │   ├── [할당]
  │   ├── [승인]
  │   ├── [거부]
  │   └── [보류]
  └── <ReportProcessModal />     # 처리 모달
</ReportDetailPage>
```

## 필터 컴포넌트

### ReportFilters

```jsx
<ReportFilters>
  <SearchInput 
    placeholder="신고 사유, 대상 검색..."
    value={search}
    onChange={setSearch}
  />
  <Select
    label="상태"
    options={['all', 'PENDING', 'IN_PROGRESS', 'RESOLVED', 'REJECTED']}
    value={status}
    onChange={setStatus}
  />
  <Select
    label="유형"
    options={['all', 'SPAM', 'HARASSMENT', 'INAPPROPRIATE', 'COPYRIGHT', 'OTHER']}
    value={type}
    onChange={setType}
  />
  <Select
    label="우선순위"
    options={['all', 'LOW', 'MEDIUM', 'HIGH', 'URGENT']}
    value={priority}
    onChange={setPriority}
  />
  <Select
    label="담당자"
    options={['all', 'me', 'unassigned']}
    value={assignedTo}
    onChange={setAssignedTo}
  />
</ReportFilters>
```
