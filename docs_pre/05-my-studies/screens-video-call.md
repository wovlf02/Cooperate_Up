## 기능

### 비디오 컨트롤

| 컨트롤 | 설명 |
|--------|------|
| 음소거 | 마이크 on/off |
| 비디오 | 카메라 on/off |
| 화면공유 | 화면 공유 |
| 종료 | 통화 종료 |

### 뷰 모드

| 모드 | 설명 |
|------|------|
| 발표자 뷰 | 발표자 크게, 나머지 작게 |
| 갤러리 뷰 | 모두 동일 크기 |

---

## 기술 스택

| 기술 | 용도 |
|------|------|
| WebRTC | P2P 미디어 스트림 |
| Socket.IO | 시그널링 |
| Signaling Server | 연결 조율 |
