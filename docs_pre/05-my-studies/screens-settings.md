## 수정 가능 항목

| 항목 | 설명 |
|------|------|
| 이모지 | 대표 이모지 |
| 이름 | 스터디 이름 |
| 카테고리 | 분류 |
| 설명 | 스터디 소개 |
| 태그 | 관련 키워드 |
| 최대 인원 | 정원 |
| 공개 여부 | 검색 노출 |
| 자동 승인 | 가입 즉시 참여 |
| 모집 상태 | 신규 멤버 모집 |
