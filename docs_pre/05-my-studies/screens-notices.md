## 기능

### 공지 표시

| 요소 | 설명 |
|------|------|
| 📌 아이콘 | 고정/중요 공지 표시 |
| 제목 | 공지 제목 |
| 조회수 | 조회 횟수 |
| 작성자 | 작성자 이름 |
| 작성일 | 작성 날짜 |

### 권한별 기능

| 기능 | MEMBER | ADMIN | OWNER |
|------|--------|-------|-------|
| 조회 | ✓ | ✓ | ✓ |
| 작성 | ✗ | ✓ | ✓ |
| 수정 | ✗ | 본인만 | ✓ |
| 삭제 | ✗ | 본인만 | ✓ |
| 고정 | ✗ | ✓ | ✓ |
